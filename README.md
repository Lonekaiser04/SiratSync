# SiratSync Website

Official landing page for SiratSync - Your Islamic Lifestyle Companion. This website showcases the features of the SiratSync app and provides download links for users.

## About SiratSync

SiratSync is a comprehensive Islamic lifestyle application designed to help Muslims strengthen their connection with Allah through daily technology. This website serves as the primary information hub, showcasing app features and providing easy access to downloads.

### Mission
"To create technology that serves the Ummah by making Islamic practices easier, more accessible, and integrated into daily life."

## Features Showcased

### App Features
- **Quran Reader**: Complete Quran with Uthmani script, translations, and bookmarks
- **Prayer Times**: Accurate prayer calculations with Qibla direction and mosque finder
- **Hadith Library**: Authentic collections from Sahih Bukhari & Sahih Muslim
- **Duas Collection**: Comprehensive supplications for daily life occasions
- **Digital Tasbih**: Beautiful counter with multiple dhikr options
- **Habit Tracker**: Build and maintain Islamic habits with streak tracking
- **Community**: Connect with Muslims worldwide in a safe space
- **Multilingual**: Support for English, Urdu, Hindi, Arabic, Malay

### Website Features
- Fully responsive design optimized for mobile, tablet, and desktop
- SEO optimized with meta tags, Open Graph, and Twitter Cards
- Fast loading with optimized images and minimal dependencies
- Interactive elements including screenshot slider and feature cards
- Accessibility features with semantic HTML
- Cross-browser compatibility
